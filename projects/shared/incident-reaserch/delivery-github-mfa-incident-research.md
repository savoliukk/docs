# Дослідження інцидентів: MFA залежить від налаштувань особистих GitHub-акаунтів

## Контекст з матриці

- Домен: CI/CD, репозиторії та доставка змін
- Зона: GitHub
- Загроза: MFA залежить від налаштувань особистих GitHub-акаунтів
- Ризик: Якщо учасник репозиторію не має достатнього захисту акаунта, компрометація його GitHub може відкрити доступ до коду або змін у репозиторіях. Для доступів власника або адміністратора це особливо критично.
- Поточна пріоритетність: Висока

## Пошуковий фокус

Шукались реальні публічні інциденти, де підтверджено або чітко описано схожий розрив у контролях чи ризиковий сценарій для теми: `MFA залежить від налаштувань особистих GitHub-акаунтів`. Пріоритет надавався офіційним післяінцидентним розборам, урядовим попередженням, регуляторним і судовим документам, звітам постачальників безпекових досліджень і надійним медіа. Усі описи нижче подані українською мовою; назви джерел, компаній, продуктів і технічних ідентифікаторів залишені в оригіналі.
