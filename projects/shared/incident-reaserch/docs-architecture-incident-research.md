# Дослідження інцидентів: Витік архітектури та внутрішніх назв

## Контекст з матриці

- Домен: Документація, код і AI-контекст
- Зона: Код і документація
- Загроза: Витік архітектури та внутрішніх назв
- Ризик: Повний доступ до репозиторіїв і документації розкриває архітектуру, внутрішні назви хостів, короткі інструкції, заплановані задачі безпеки і правила іменування секретів. Це допомагає зловмиснику швидше знайти слабкі місця.
- Поточна пріоритетність: Середня

## Пошуковий фокус

Шукались реальні публічні інциденти, де підтверджено або чітко описано схожий розрив у контролях чи ризиковий сценарій для теми: `Витік архітектури та внутрішніх назв`. Пріоритет надавався офіційним післяінцидентним розборам, урядовим попередженням, регуляторним і судовим документам, звітам постачальників безпекових досліджень і надійним медіа. Усі описи нижче подані українською мовою; назви джерел, компаній, продуктів і технічних ідентифікаторів залишені в оригіналі.

## Висновок

- Повторювані патерни: Secret scanning/repository controls не виявили ключ доступу у публічному репозиторії протягом років. ; Source-code/document repository розкриття розкрив внутрішній код і бізнес-контекст. ; Збережені OAuth-токени й конфігураційні змінні конвеєра доставки створили широкий радіус ураження між платформою як сервісом (PaaS), GitHub і CI/CD..
- Наскільки поширений тип проблеми: знайдені кейси показують, що цей клас розриву трапляється в реальних інцидентах із SaaS-сервісами, хмарними середовищами, CI/CD, пристроями або процесами управління безпекою; точна поширеність не оцінювалась кількісно.
- Чи варто залишити поточну пріоритетність: поточну пріоритетність `Середня` варто залишити, якщо внутрішній стан контролів відповідає описаному ризику; для непрямих кейсів це слід трактувати як підтримувальний, а не абсолютний доказ.
- Короткий текст для колонки "Пов'язані інциденти": [Twitch - source code and payout data leak](docs-architecture-incident-research.md): Twitch 2021 показує, що витік репозиторію може розкрити вихідний код і внутрішній бізнес-контекст для ризику `Витік архітектури та внутрішніх назв`.
