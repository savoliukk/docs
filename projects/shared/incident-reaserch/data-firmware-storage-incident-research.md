# Дослідження інцидентів: Підміна firmware або файлів

## Контекст з матриці

- Домен: Дані, бази даних і сховища файлів
- Зона: MinIO / файли / firmware (прошивки)
- Загроза: Підміна firmware або файлів
- Ризик: Доступ до firmware, файлового сховища або конфігурації розгортання може дозволити підміну firmware, розповсюдження шкідливих файлів, витік вкладень або компрометацію клієнтів через офіційний шлях завантаження.
- Поточна пріоритетність: Висока

## Пошуковий фокус

Шукались реальні публічні інциденти, де підтверджено або чітко описано схожий розрив у контролях чи ризиковий сценарій для теми: `Підміна firmware або файлів`. Пріоритет надавався офіційним післяінцидентним розборам, урядовим попередженням, регуляторним і судовим документам, звітам постачальників безпекових досліджень і надійним медіа. Усі описи нижче подані українською мовою; назви джерел, компаній, продуктів і технічних ідентифікаторів залишені в оригіналі.

### 1. ASUS Live Update - Operation ShadowHammer

- Статус дедуплікації: повний первинний опис інциденту подано в [delivery-gitops-persistence](delivery-gitops-persistence-incident-research.md); тут наведено тільки стислий зв'язок із поточним ризиком.
- Рік і дата інциденту або публікації: 2019
- Тип джерела: звіт постачальника про загрозу
- Джерела:
  - [Kaspersky ShadowHammer report](https://securelist.com/operation-shadowhammer/89992/)
- Стислий опис: ASUS ShadowHammer показує ризик підміни шляху оновлення firmware/ПЗ.
- Використаний розрив у контролях або ризиковий розрив у цьому рядку: Підписаний механізм оновлення став каналом для вибіркової доставки шкідливого коду.
- Наслідки для цього контексту: Інцидент поставив під сумнів довіру до механізмів оновлення firmware/ПЗ.
- Релевантність до цієї загрози/ризику: Релевантність пряма: кейс підтверджує або ілюструє розрив у контролях або ризиковий розрив для загрози 'Підміна firmware або файлів' у зоні 'MinIO / файли / firmware (прошивки)', бо показує реальний наслідок схожого розриву: Підписаний механізм оновлення став каналом для вибіркової доставки шкідливого коду.
- Рівень доказовості: середній
- Чи підсилює пріоритетність ризику: так
- Згадки про інструменти безпеки, якщо є: ASUS Live Update, code signing
