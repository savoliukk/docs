# Дослідження інцидентів: Неповний список внутрішніх сервісів, доступних з інтернету

## Контекст з матриці

- Домен: Мережевий і приватний доступ
- Зона: Внутрішні панелі
- Загроза: Неповний список внутрішніх сервісів, доступних з інтернету
- Ризик: Без повної інвентаризації розкриті services складно зрозуміти, які панелі або API доступні з untrusted network. Непомічений сервіс може стати точкою входу.
- Поточна пріоритетність: Середня

## Пошуковий фокус

Шукались реальні публічні інциденти, де підтверджено або чітко описано схожий розрив у контролях чи ризиковий сценарій для теми: `Неповний список внутрішніх сервісів, доступних з інтернету`. Пріоритет надавався офіційним післяінцидентним розборам, урядовим попередженням, регуляторним і судовим документам, звітам постачальників безпекових досліджень і надійним медіа. Усі описи нижче подані українською мовою; назви джерел, компаній, продуктів і технічних ідентифікаторів залишені в оригіналі.

- Повторювані патерни: Internet-facing VPN appliance став точкою входу до внутрішніх мереж. ; Kubernetes/control-plane інтерфейс був доступний без належної автентифікації. ; Внутрішня база support analytics була помилково доступна з інтернету..
- Наскільки поширений тип проблеми: знайдені кейси показують, що цей клас розриву трапляється в реальних інцидентах із SaaS-сервісами, хмарними середовищами, CI/CD, пристроями або процесами управління безпекою; точна поширеність не оцінювалась кількісно.
- Чи варто залишити поточну пріоритетність: поточну пріоритетність `Середня` варто залишити, якщо внутрішній стан контролів відповідає описаному ризику; для непрямих кейсів це слід трактувати як підтримувальний, а не абсолютний доказ.
- Короткий текст для колонки "Пов'язані інциденти": [Microsoft - exposed customer support database](network-public-unknown-incident-research.md): Microsoft 2020 показує, що помилково налаштована база технічної підтримки може розкрити клієнтський контекст без окремої атаки на застосунок для ризику `Неповний список внутрішніх сервісів, доступних з інтернету`.
